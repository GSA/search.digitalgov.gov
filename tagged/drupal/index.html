usasearch Drupal Module Help Documentation

To use the Search.gov service on a Drupal-based website, you should install the usasearch Drupal module (External link). It is possible to manually modify Drupal to connect to our service, but the module allows you to minimize the development work required for ongoing use and maintenance.
